## Installation and setup

âš ï¸ First, make sure you have: [Node](https://nodejs.org/dist/latest-v12.x/) plus [Yarn](https://classic.yarnpkg.com/en/docs/install/) and [Git](https://git-scm.com/downloads)

```text
git clone https://github.com/austintgriffith/scaffold-eth.git
cd scaffold-eth
git checkout defi-subgraphs
yarn install
yarn chain
```

> in a second terminal window:

```text
cd scaffold-eth
yarn start
```

> in a third terminal window:

```text
cd scaffold-eth
yarn deploy
```

If everything goes smoothly you should now have a local network running, with the starter contracts deployed and the frontend React app running on [https://localhost:3000](https://localhost:3000/).
